# Как запустить

Надо собрать и запустить проект docker compose

```shell
docker compose up -d
```

Если все ОК, нодо сконфигурировать проект и заполнить mongodb данными. Для этого надо запустить скрипт из директории scripts.
Обратите внимание, что в скрипте есть задержки. Дождитесь надписи **"Press enter to continue"** которая означает конец выполнения скрипта.

```shell
./scripts/mongo-init.sh
```

# Как проверить

Откройте в браузере [http://localhost:8080](http://localhost:8080)
