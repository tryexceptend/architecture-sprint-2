# Задание 1

## Запуск

1. Запуск mongodb и приложения

```shell
docker compose up -d
```

2. Скрип конфигурации mongjdb

```shell
./scripts/mongo-init.sh
```

## Как проверить

### Если вы запускаете проект на локальной машине

Откройте в браузере http://localhost:8080
